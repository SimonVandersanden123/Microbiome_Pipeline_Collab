# Microbiome Pipeline Collab

A standardized pipeline for processing soil microbiome data from the Griftpark project.

## 🚀 Quick Start
1. Clone the repo.
2. Open `Microbiome_Pipeline_Collab.Rproj` in RStudio.
3. Open `Pipeline_Overview.Rmd` and run all chunks.

## 📖 Documentation
For a detailed explanation of the normalization (BMR/CLR) and diversity metrics, see:
[Primary Documentation](./Primary_Documentation.md)

## 📁 Structure
- `/scripts`: Numbered R scripts for the analysis steps.
- `/data`: Input Phyloseq objects and metadata.
- `/results`: here plots will hopefully appear in the future
